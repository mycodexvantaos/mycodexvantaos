## ❓ 必須確認的 4 大問題

**進入設計階段前需逐項確認**：

### Q-1: 同步自動化?
目前假設：Push to main 自動觸發同步  
**需確認**：是否正確？是否需手動選項？

### Q-2: 回滾策略?
目前假設：GKE 自動回滾，Supabase 不回滾  
**需確認**：各平台 SLA 是什麼？統一 API 必要嗎？

### Q-3: 策略版本控制?
目前假設：OPA 版本號與 git tag 對齊  
**需確認**：新策略需版本號嗎？棄用流程是什麼？

### Q-4: 多租戶隔離?
目前假設：暫不支持，全組織共享  
**需確認**：是否需要 team-level workflows？
